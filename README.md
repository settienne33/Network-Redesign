Network Redesign 

A redesigned multi-building network architecture centered on improving security, reliability, and performance through modern topology, monitoring, and access controls.


Implementation of a mesh topology to avoid single points of failure and increase data reliability.

Added network observability to enhance performance monitoring and troubleshooting.

Introduced a firewall to protect against internal and external threats.

WLAN deployed for mobility, reduced cabling, and increased collaboration across departments.


Mesh topology, WLAN, firewalls, LAN/WAN, network observability, secure architecture.
